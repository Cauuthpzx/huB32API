# hub32api REST API — v2 Reference

Base URL: `http://host:11081/api/v2`

v2 is a superset of v1. All v1 endpoints remain available.

---

## New in v2

### POST /batch/features
Apply a feature to multiple computers atomically.

**Request:**
```json
{
  "computerIds": ["uuid1", "uuid2"],
  "featureUid": "ccb535a2-1d24-4cc1-a709-8b47d2b2ac79",
  "operation": "start",
  "arguments": {}
}
```

**Response 200:**
```json
{
  "total": 2, "succeeded": 2, "failed": 0,
  "results": [
    { "computerId": "uuid1", "success": true },
    { "computerId": "uuid2", "success": true }
  ]
}
```
