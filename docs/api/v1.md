# veyon32api REST API — v1 Reference

Base URL: `http://host:11081/api/v1`

All endpoints except `POST /auth` require `Authorization: Bearer <token>`.

---

## Authentication

### POST /auth
Login and receive JWT token.

**Request:**
```json
{ "method": "veyon-key", "keyName": "default", "keyData": "<pem>" }
```

**Response 200:**
```json
{ "token": "eyJ...", "tokenType": "Bearer", "expiresIn": 3600 }
```

---

## Computers

### GET /computers
List all computers.

**Query params:** `?location=ClassroomA` `?state=online` `?limit=50` `?offset=0`

**Response 200:**
```json
{
  "computers": [
    { "id": "uuid", "name": "PC-01", "hostname": "pc01.lan", "location": "Room A", "state": "online" }
  ],
  "total": 1
}
```

### GET /computers/:id
Get single computer.

### GET /computers/:id/framebuffer
Get screenshot.

**Query params:** `?width=800&height=600&format=jpeg&quality=85`

**Response 200:** Binary image (`Content-Type: image/jpeg`)
